# CS50's Introduction to Artificial Intelligence with Python

This repository contains my solutions and implementations for Harvard University's **CS50's Introduction to Artificial Intelligence with Python (CS50 AI)**.

## Course Overview

CS50 AI introduces the fundamental concepts and algorithms that power modern artificial intelligence. Throughout the course, students explore both classical and modern AI techniques, building practical projects that apply these concepts to real-world problems.

Topics covered include:

- Search Algorithms
- Knowledge Representation
- Logical Inference
- Probability and Uncertainty
- Optimization
- Machine Learning
- Reinforcement Learning
- Neural Networks
- Natural Language Processing
